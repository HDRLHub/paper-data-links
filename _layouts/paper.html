Usage

  1. Python Data Download Script: Run the download_script.py to fetch data related to the paper. Ensure you have all the necessary dependencies, including SunPy, installed. This script is designed to retrieve data specifically from the VSO. Please note that this script represents a starting point and may not be 100% perfect. Users are encouraged to review and modify it as needed.
  2. Instrument Information: Check INSTRUMENT_DETAILS.md for comprehensive information on the instruments used. This can provide context and help in understanding the data fetched by the script.
